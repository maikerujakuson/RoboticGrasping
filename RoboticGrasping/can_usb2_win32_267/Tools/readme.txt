                CAN-USB/2, CAN-USB/Micro, CAN-AIR/2, CAN-CBX-AIR/2
                
This folder contains the following additional tools for CAN interfaces
attached to the USB on Windows. All tools are 32-bit binaries which
also work without any restrictions on 64-bit versions of Windows.

Firmware Update:

updusb2292.exe   - Firmware update tool for the CAN-USB/2 (FW 1.1.0)
updcanmicro.exe  - Firmware update tool for the CAN-USB/Micro (FW 1.1.5)
updcanair2.exe   - Firmware update tool for the CAN-AIR/2 (FW 1.1.4)
updcbxair2.exe   - Firmware update tool for the CAN-CBX-AIR/2 (FW 1.1.4)

Configuration:

canairconfig.exe - Configuration tool for the CAN-AIR/2 V1.3
